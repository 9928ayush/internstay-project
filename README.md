# InternStay 

> A peer-to-peer marketplace for college students to find short-term, move-in-ready PG and hostel accommodations for summer internships.

## 📖 About the Project

InternStay makes it easy for students to secure housing near tech parks and corporate offices. By offering a streamlined booking marketplace, it connects property hosts with young professionals looking for verified, short-term stays with essential amenities like high-speed WiFi and power backup.

## 🚀 Features

* **Secure Login:** Quick and safe user authentication.
* **Live Availability:** Real-time updates on property bookings.
* **Two-Sided Platform:** Seamless experience for both hosts and students.
* **Verified Amenities:** Focus on properties ready for remote/hybrid work.

## 🛠️ Tech Stack

* **Frontend:** React
* **Backend & Database:** Firebase (Cloud Firestore & Authentication)

## ⚙️ Getting Started

To run this project locally, simply clone the repository and start the development server.

```bash
# Clone the repository
git clone [https://github.com/your-username/InternStay.git](https://github.com/your-username/InternStay.git)

# Navigate into the directory
cd InternStay

# Install dependencies
npm install

# Start the application
npm start
```
